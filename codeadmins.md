출석부 모듈 코드 운영방침.
==================

개요
---
* 출석부 모듈의 코드 공헌을 받을 때 기본적인 규칙을 정하여 공통된 관리를 위함입니다.

공통
---
* 들여쓰기는 항상 탭을 사용하도록 합니다. (띄어쓰기가 4번들어가는 탭은 금지)

1. PHP파일

* 기본 `if문, 반복문 등` 은 기본적으로 대갈호를 한 줄 아래로 내려서 대갈호를 엽니다.

```php

if($회사 == 'SMent')
{
	if($contest->sosi == 'Y')
	{
		$context->sosi = $obj->sosi
	}
}

for($1,$2,$3)
{
	$s1 = 'sosi';
}

```
