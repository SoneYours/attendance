att_bootstrap
=============

부트스트랩 출석부스킨

개요
----

이 소프트웨어는 '부트스트랩 출석부 스킨' 입니다.

이 소프트웨어는 GPL v2을 채택한 오픈 소스 소프트웨어로써 자유롭게 사용할 수 있는 자유 소프트웨어 입니다.
이 라이선스 조건을 준수하는 조건하에 누구나 자유롭게 사용할 수 있습니다.

주요 기능
---------
* XE 출석부 소프트웨어에서 기본적으로 지원하는 스킨을 좀더 개량된 버전으로 쓸 수 있음.

참고 사항
---------
버전명 형태를 기존 0.0.1 에서 0.0.0.1 형태로 바꾸었습니다

0.0.0.1에서 첫번째 0은 테마스킨을 가리킵니다. 만약 대규모패치로 인해 디자인의 형태가 아주 많이 바뀔경우 이 자리의 숫자가 바뀌게 됩니다.

두번째와 세번째의 숫자는 조금의 디자인의 추가라던지 기능추가가 이루어질경우 2자리가 10진법의 형태로 올라갑니다. ( 0.0.9.0 에서 해당 부분의 디자인이 추가되거나 문제해결이 이루어질경우 0.1.0.0 형태로 바뀌게 됩니다. 또한 0.9.9.0 에서 바뀌게 될경우 테마스킨 부분의 숫자는 건들지 않은체 0.10.0.0 으로 바뀌게 될 것입니다.)

마지막 자리의 4번째 숫자는 마이너 패치입니다.
소소한 버그, 간단한 버그 등등을 패치하여 디자인혹은 기능이 빠지거나 추가 되는 것 없이 패치 될경우 이자리의 숫자를 바뀌게 됩니다.

위 사항은 0.0.3.1 버전 다음버전부터 이루어집니다..(0.0.3.1 은 제가 귀찮앗나 본지..그냥 뒤에 1을 추가 하고 배포 했네요..-긁적.. 나도 사람인지라 ㅠㅠ-
